# 为什么分代
不同对象生命周期不同，让这些不同生命周期的对象分代存放，并各自使用合适的回收算法，可以大大提高垃圾回收的效率


不分代的话，整个堆包含新建的对象和生命周期长的对象，每次垃圾回收都是对整个堆空间进行回收，花费时间相对会长；
同时，因为每次回收都需要遍历所有存活对象，对于生命周期长的对象而言，这种遍历是没有意义的，因为可能进行了很多次遍历，但是他们依旧存在

所以分代各自存放
