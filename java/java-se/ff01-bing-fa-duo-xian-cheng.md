# A,B,C 三个线程顺序执行

#### join()方法 （start+join）
```java
Thread t1 = new Thread(new Runner("A"));  
Thread t2 = new Thread(new Runner("B"));  
Thread t3 = new Thread(new Runner("C"));  
t1.start();  
t1.join();  
t2.start();  
t2.join();  
t3.start();  
t3.join();  
```

#### 线程run()内join
```java
Thread t1 = new Thread(new Runner("A"));  
Thread t2 = new Thread(new Runner("B"));  
Thread t3 = new Thread(new Runner("C"));  
t1.start();  
t1.join();  
t2.start();  
t2.join();  
t3.start();  
t3.join();  
```
